# 诗词自动生成系统

## 一、项目背景

中国古代诗词读起来朗朗上口，既文雅又能表达不同意境。近年来人工智能与文学艺术的结合日趋紧密，AI 自动绘画、自动作曲等方向都成为研究热点。诗歌自动生成是一项有趣且具有挑战性的任务。

本系统通过深度学习技术，根据给定的场景或关键字自动生成唐诗。在此基础上可进一步扩展，支持根据图片或给定场景自动生成汉语作文。

## 二、问题定义

基于 NLP 技术进行文本分析与诗词生成，核心任务包括：

- 根据给定语句进行文本分析，寻找关键词
- 基于 LSTM 算法完成自动诗词生成
- 基于无监督学习实现诗词生成
- 生成诗词在其他指标（流畅性、通顺性等）上无损失或尽量减小损失

## 三、功能需求

| 编号 | 功能 | 说明 |
|------|------|------|
| F1 | 关键字生成诗词 | 给定一个关键字，自动生成符合韵律要求的诗词 |
| F2 | 场景生成诗词 | 输入一句场景描述，进行文本分析和情感分析，根据分析结果自动生成诗词 |
| F3 | 图片生成诗词 | 输入一幅图片，进行图像识别，根据识别内容生成诗词 |

## 四、技术指标

### 4.1 核心技术

- **NLP 自然语言处理**：文本分词、关键词提取、语义理解
- **文本分析**：句法分析、主题建模
- **情感分析**：情感极性判断、情感强度量化
- **图像分析**：图像分类、物体检测、场景识别
- **RNN / LSTM 网络**：序列生成模型
- **Python 深度学习框架**：PyTorch / TensorFlow

### 4.2 性能指标

- 生成诗词符合古典诗词格律（平仄、押韵、对仗）
- 输出结果流畅通顺，语义连贯
- 内容与输入关键词/场景/图片相关
- 单次生成响应时间 ≤ 3 秒

## 五、项目约束

- 开发语言：Python
- 深度学习框架：PyTorch 或 TensorFlow
- 版本控制：Git（GitHub）
- 平台：Web 应用或桌面应用
